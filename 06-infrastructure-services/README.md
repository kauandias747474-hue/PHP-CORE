# 🌐 Infrastructure Services

Módulo de integração com o mundo externo e serviços de suporte ao sistema.

### 🛠️ Funcionalidades:
- **Logger:** Registro de eventos do sistema em arquivos ou logs externos.
- **Mailer:** Interface para envio de e-mails transacionais.
- **File Storage:** Abstração para upload e manipulação de arquivos de forma segura.
